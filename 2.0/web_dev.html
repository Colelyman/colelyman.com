Types of Web Sites I Design

There are two main types of websites I am willing design, a custom website or a website from a CSS template. There are advantages and disadvantages to both of the types of sites.

Custom Websites

One advantage to a custom website is that it is completley your own. One disadvantage to a custom built site is that it will take longer to develop, and will also have a induce a greater cost compared to a CSS template website.
